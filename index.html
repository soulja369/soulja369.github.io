FamilyTask-App is a family chore and task management system built as a web app, installable PWA, and embeddable Home Assistant dashboard view. The system uses progressive disclosure: Basic Mode is fully functional as a standalone product, and Advanced Mode (points, rewards, approvals, photo proof) is opt-in — no one is forced into the complex flow.

RewardStack
RewardStack is a consumer app for managing every rewards and loyalty program a person belongs to. The north star is a smart assistant that knows what you have ("80k Chase points, 40k Marriott points") and tells you how to use it ("book this trip cheaper with points"). The honest MVP is smaller: a directory and launcher — find programs, organize the ones you use, open them fast. Everything richer is earned on top of that foundation.
